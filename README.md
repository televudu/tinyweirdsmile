# Tiny Weird Smile - Visor de PDF

Este proyecto es una página web muy simple dedicada únicamente a mostrar un PDF largo y permitir que los usuarios lo recorran con scroll.

## Uso

1. Copia tu archivo PDF dentro de la carpeta `assets/` y renómbralo como `documento.pdf`.
2. Abre `index.html` en tu navegador preferido.
3. El PDF aparecerá en pantalla completa y podrás desplazarte con el scroll. Si el navegador no puede renderizarlo, se mostrará un enlace directo para descargarlo.

Si prefieres mantener otro nombre para el archivo, ajusta la ruta del atributo `data` dentro de `index.html` para que apunte al nuevo archivo.

## Despliegue

Puedes hospedar esta página en cualquier servicio de hosting estático (GitHub Pages, Netlify, Vercel, etc.). Solo asegúrate de subir también el PDF dentro de la carpeta `assets/`.
